# Logos de fallback por medio

Esta carpeta contiene imágenes de fallback específicas por dominio cuando no se puede extraer una imagen del artículo.

## Archivos necesarios

- `canarias7.png` - Logo de Canarias7
- `laprovincia.png` - Logo de La Provincia
- `cabildo.png` - Logo del Cabildo de Fuerteventura
- `radioinsular.png` - Logo de Radio Insular
- `fvdigital.png` - Logo de Fuerteventura Digital
- `ondafv.png` - Logo de Onda Fuerteventura
- `newsdata.png` - Logo de newsdata.io

## Dimensiones recomendadas

- Ancho: 1200px
- Alto: 630px (ratio 1.91:1 óptimo para Open Graph)
- Formato: PNG con transparencia o JPG

## Fallback genérico

Si no existe un logo específico para un dominio, se usa `images/Fuerteventura.jpeg`.
